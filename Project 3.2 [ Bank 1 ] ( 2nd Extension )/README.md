# 🏦 Bank Clients & User Management System (C++)

# 🏦 [ Bank 1 ] (2nd Extension)

## 📌 Overview
A simple program to manage bank clients and users. It saves data in files and controls who can use each part of the system.

---

## 🚀 Main Features
- **Client Tasks**: Add, show, edit, and delete client info.
- **Money Tasks**: Deposit and withdraw money with balance checks.
- **User Tasks**: Add and manage users who use the system.
- **Permissions**: Set what each user is allowed to do.
- **Login Screen**: Protects the system with a username and password.

---

## ⚙️ How it Works
- **File Saving**: Keeps all data safe in text files.
- **Permissions**: Uses simple code to check user rights.
- **Data Groups**: Keeps clients and users data organized separately.
- **Table View**: Shows data in clean, organized tables.

---

## ⚠️ Challenges & Solutions
- **Security**: Blocked users from opening screens without permission.
- **Admin Protection**: Made it impossible to delete the main "Admin" account.
- **Errors**: Stopped the program from using duplicate account numbers.
- **Saving**: Files update immediately after any change.

---

## 🎯 What I Learned
- How to make a login system for a program.
- How to give different powers (permissions) to users.
- How to work with many files at the same time.
- How to build a professional menu system.

---

## 💼 Project Purpose
To show how to build a safe and organized system for a small business like a bank.
